# Teams — progress tracker

Each team entry should include:

* **Current stage** (Ground, Define, Prototype, Test, Reflect)

* **Short 1-line status update** (what did you do this week?)

* **Required deliverable for next check-in** (e.g., 3 photos + updated submission.md)

* **Next meeting date/time and contact**

## Example Status Line

Team Waste — Stage: Prototype — This week: installed one collection drum, tested for 48 hours. Next deliverable: 48-hour weight log + 3 photos + user quote.
