# Edix — Final Solution

## Edix

**"Turning school data into sovereign student assets."**

## The System

### High-Performance Engine
| Component | Technology | Function |
|-----------|------------|----------|
| Backend | FastAPI (Python) | Async speed |
| AI Core | Google Gemini | Grading automation |
| Jobs | Celery + Redis | Background processing |
| Database | PostgreSQL | Data persistence |

### Document Engine
*   Generates paper reports for parents
*   Bridges digital and physical expectations

### Trust Layer (Invisible)
*   **Atala PRISM** — SSI credentials
*   **PyCardano** — On-chain minting
*   **Twilio** — SMS to parents

## The Flow
```
Teacher finalizes grade
    ↓
VC automatically generated
    ↓
School DID signs credential
    ↓
Student receives in wallet
```

## Live Status
✅ MVP demonstrating VC issuance upon grade finalization
